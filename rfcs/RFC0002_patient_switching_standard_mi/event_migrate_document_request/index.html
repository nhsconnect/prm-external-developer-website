RFC0002 Patient Switching Standard MI

Migrate Document Request

Event Description

The "Migrate Patient Record Bundle" response must contain a list of all documents that the Requesting System must attempt to retrieve. The Requesting System must attempt to retrieve all documents that are under 100mb and of supported MIME types. The Migrate Document Request event should be sent immediately after the Requesting System makes an individual request to retrieve each document via GP Connect using the ‘Migrate Document’ end point.

Top Level Event Fields

FieldDescription
eventIdUnique identifier for this event.
eventGeneratedDateTimeThe ISO 8601 Date time in UTC of when the event was generated by the system.
registrationIdThe unique identifier for this registration that can be resolved in the clinical system.
reportingSystemSupplierThe system supplier who generated the event. A unique identifier given to you as a supplier by NHSD.
reportingPracticeOdsCodeThe ODS code of the practice generating the event.
payloadAn object that contains the detailed payload of the event.

Payload Event Fields

FieldDescription
registrationAn object that contains information about the registration process
gpTransferMetadataAn object that contains information about the transfer
attachmentInformation about the attachment this documentation request is attempting to receive.

Registration Event Fields

FieldDescription
requestingPracticeOdsCodeThe ODS code of the practice requesting the EHR.
sendingPracticeOdsCodeThe ODS code of the practice sending the EHR.

GP Transfer Metadata Event Fields

FieldDescription
conversationIdThe ConversationID used in the transfer process for this registration.
transferEventDateTimeThe ISO 8601 Date time in UTC of when the document migration was requested.

Attachment Event Fields

FieldDescription
attachmentIdA unique id for the attachment that can be resolved in the clinical system.
clinicalType(Optional) The text value corresponding to the value used in EhrAttachmentCode in the code element of ExternalDocument as defined in the MIM.
mimeTypeThe mime type of the attachment as used as part of the text element in ExternalDocument as defined in the MIM.
sizeBytesThe size in bytes of the attachment.
